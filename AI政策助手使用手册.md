## 三、输出格式选择

### 3.1 简洁版（适合快速浏览）

```
指令示例：
"简要汇总本周政策"
"用列表形式展示本月重要政策"

输出特点：
- 只显示政策标题和核心要点
- 按重要程度排序
- 标注紧急程度
```

### 3.2 详细版（适合深入研究）

```
指令示例：
"详细分析国资委考核政策"
"完整解读住建部最新文件"

输出特点：
- 包含政策全文要点
- 详细的影响分析
- 关联政策梳理
- 应对建议
```

### 3.3 表格版（适合对比分析）

```
指令示例：
"用表格对比近三年考核政策"
"整理各部委关于安全生产的政策对比"

输出特点：
- 结构化展示
- 便于横向对比
- 关键差异标注
```

## 六、常见问题

### Q1：政策信息来源可靠吗？
**A**：所有政策信息均来自官方网站，包括国务院国资委、各部委官网、中国政府网等权威渠道。重要政策会提供原文链接供您核实。

### Q2：信息更新频率如何？
**A**：
- 国务院国资委、国务院政策文件库：实时监测
- 主要部委（发改委、住建部等）：每日更新
- 其他信息源：每周更新

### Q3：如何确保不遗漏重要政策？
**A**：
- 系统会按重要程度自动排序
- 紧急政策会特别标注
- 支持设置关注主题，相关新政策自动提醒

### Q4：可以查询历史政策吗？
**A**：可以。目前支持查询近3年的政策文件，且历史数据持续补充中。

### Q5：政策分析的专业性如何保证？
**A**：
- 基于大语言模型的政策理解能力
- 结合建筑央企行业知识
- 提供多维度分析框架
- 分析结果供参考，重要决策建议结合专业判断

## 七、使用建议

### 7.1 日常使用建议

1. **养成每日查看习惯**
   - 建议每天上午查看前一天的政策更新
   - 重点关注国资委和主管部委的政策

2. **建立关注清单**
   - 明确您最关心的政策主题
   - 定期调整关注重点

3. **做好信息归档**
   - 重要的政策分析及时保存
   - 建立部门内部共享机制

### 7.2 提高效率的技巧

1. **使用精准关键词**
   - 机构名称：国资委、住建部、发改委
   - 业务关键词：考核、招投标、安全生产、资质

2. **善用组合查询**
   - 同时限定时间和主题
   - 多机构对比查询

3. **递进式深挖**
   - 先获取概览，再深入细节
   - 从政策到影响再到建议

### 7.3 注意事项

1. **重要政策核实原文**
   - AI助手提供的信息供参考
   - 重要决策前建议查看政策原文

2. **关注政策时效**
   - 注意政策的发布日期和生效日期
   - 关注政策的废止和修订情况

3. **结合企业实际**
   - 政策分析需结合企业具体情况
   - 建议与相关部门沟通确认
